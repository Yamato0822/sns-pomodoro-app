# 要件定義書 (REQUIREMENTS.md)

## 1. アプリケーション名
SNS Pomodoro App

## 2. 開発の背景・目的
大学生などの学習者において、長時間の作業による集中力の低下や、孤独な作業によるモチベーション維持の難しさが課題となっている。
本アプリは「ポモドーロ・テクニック」による時間管理と、「SNS機能」による緩やかな社会的プレッシャーを組み合わせることで、ユーザーの生産性を向上させ、継続的な学習・作業を支援することを目的とする。

## 3. ターゲットユーザー
* 勉強や作業に集中したい大学生・社会人
* 孤独な作業だとサボってしまいがちな人
* 友人と互いに監視し合いながら作業したい人

## 4. 機能要件
* **ポモドーロタイマー機能:** 25分間の集中と5分間の休憩、アニメーション演出
* **SNS連携機能:** 作業完了時に成果をシェアする機能
* **ユーザー管理:** 認証機能の実装
* **統計機能:** 集中時間の履歴表示

## 5. 技術スタック
* Frontend: React Native (Expo SDK 54)
* Styling: NativeWind
* Deployment: Vercel / Netlify
